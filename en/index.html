CyberRepublic ETH Task ForceTeam for building Elastos Ethereum Sidechain Ecosystem

ETH Task Force is setup in July 2019 which dedicated to initiatives involving Elastos and Ethereum. As the Elastos ETH Sidechain goes live, there is a need to start building up the developer documentation, tutorials, demo apps and business development with other Ethereum projects.

Till now, ETH Task Force has built/port several key infrastructure services for ETH Sidechain. We are happy to see more developers get involved into the community and using our services.

Blockchain Explorer

Able to view information of transaction, block and smart contract

Testnet Faucet

Get free test token to facilitate your development

ELA Metamask

Modified Metamask that added sidechain rpc without need to add them by yourself

What we built

JSON RPC ➡ Testnet: https://rpc.elaeth.io, Mainnet: https://mainrpc.elaeth.io

MetaMask ➡ Brings Ethereum to your browser. Chrome / Firefox / Opera
